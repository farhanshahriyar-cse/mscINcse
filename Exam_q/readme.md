ABOUT THIS FOLDER
================================================================

This folder contains exam-related question papers for MSc CSE
courses offered at HSTU. Materials are collected session-wise
for the years 2024 and 2025.

Each file may include:

- Quiz Questions
- Assignment Questions
- Mid-Term Exam Questions
- Final Exam Questions

Sessions:
S1 = January - June
S2 = July - December
