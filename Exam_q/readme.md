## 🤝 Invitation to Contribute — BSc & MSc Students and Alumni

> **Do you have question papers, quiz sheets, assignment questions, or any exam-related materials for BSc CSE or MSc CSE at HSTU that are NOT yet in this repository?**

Your contribution can help **hundreds of current students** prepare better for their exams!

### How to Contribute

1. **Fork** this repository on GitHub
2. **Add** your materials (PDFs) to the relevant folder
3. **Submit a Pull Request (PR)** with a short description of what you added
4. Your PR will be **reviewed and merged** if relevant ✅

### What We Welcome

- 📄 Exam question papers (any year, any session)
- 📝 Quiz and assignment question sheets
- 🎓 **BSc CSE** question papers — any semester or year
- 🎓 **MSc CSE** question papers not yet listed here
- 🔄 Missing sessions of already listed courses
